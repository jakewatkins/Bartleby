# Bartleby, the Scrivener

I want to create an app to help me write posts for X (fka twitter).  X only shows the first 280 characters of a post.  If the post is longer than 280 characters it hides everything after 280 and places a "show more" link to expand the content.
I want to create a simple editor that when the user exceeds 280 characters the app inserts and asterix (*) at the 280 character so the user can see where their content would be cut.  As the user edits the content the app will keep the asterix at the 280 charater spot unless they make their post shorter than 280.

I think we should implement this as an obsidian plugin.
